# Dolphin Theme

Navy/gold desk theme and navigation for the Dolphin International ERPNext site.

Applies globally on every desk page via `app_include_css` / `app_include_js`:
- Navy `#0F2540` sidebar with gold `#D4A24A` active item
- Gold Edit/Print/Home buttons, grey Back, gold-accent tabs and card borders
- Breadcrumb Home icon rerouted to the Dolphin workspace
- Floating "Back to Workspace" button

Install on a Frappe Cloud private bench by adding this repository as a custom app, then installing it on the site.
